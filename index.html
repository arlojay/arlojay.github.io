My projects

This is a collection of some of the projects I've worked on in the past. Hopefully they still work, unlike my previous projects hosted on Replit which have been lost to the void of pseudocurrency and their everlasting desire to become their own (self-centered) platform.

Like many other people, I start and stop projects, many of which get abandoned. I'm getting better about that now. My philosophy is that side projects hone your skills. You may feel like you get nothing out of working on a project if it doesn't get complete, but you learn many new skills that you'll master through consecutive projects. For example, if you figure out through trial and error that shared buffers aren't synchronous across threads, you'll design your next project with that in mind.

Cosmic Reach Dynamic Asset Generator

Cosmic Reach Dynamic Asset Generator is a library written in TypeScript that allows for easy creation of Cosmic Reach mods.

I made this project to accompany development of Immersive Cables, another project I made.

The previous version of this project (released at arlojay/cosmic-reach-modding-api) worked for old versions of Cosmic Reach, but was limited in its application to future versions. It had little expandability, and was designed for use only by me. So I made this, CRDAG.

If you're a Cosmic Reach player, feel free to use this tool to make your own mods!

Immersive Cables

Immersive Cables is a mod for Cosmic Reach that adds cables, logic gates, and utility blocks.

Technically, this mod is a second edition of a previous Immersive Cables. The first was created as a demo mod for what was possible with data-modding. It was done with an additionally first-edition version of CRDAG, then called CRMAPI. I remade both projects for modern versions of Cosmic Reach because the old versions were growing to be unmanageable and outdated.

You can find the wiki at arlojay/immersive-cables-wiki!

The mod is currently closed-source, but I have plans to eventually make it public. I'm not sure when.

Melvinture

Melvinture is a fan project based on @sirninjamonkey's Melvin, which was created as a suggestion for Cosmic Reach.

Melvinture is a game made in Construct 3, inspired by a piece of concept art for a Melvin-centric game. When I saw Ninja's image, I thought it would be interesting to actually turn it into a game.

I didn't originally mean for the game to become its entire own thing; I mostly made it as a mediocre spin-off. I'm currently working on making it into a fully-fledged game with multiple chapters.

You can play the current game at arlojay.com/melvinland!
